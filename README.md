# chat-A-chat-application
A chat application using socket.io and express.js.

### Visit : https://chat-dorm.herokuapp.com/

### Tools-Used
* Node.js
* socket.io
* express.js

### Features
* Room based
* Multicast,Unicast Support inside chat room
* Join any room using room name
* Image sending support
